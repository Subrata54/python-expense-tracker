# Expense Tracker

A simple command-line Expense Tracker built with Python.

## Features

* Add expenses
* View all expenses
* Calculate total expenses
* Simple menu-driven interface
* Beginner-friendly Python project

## How It Works

1. Add an expense by entering:

   * Date
   * Product Name
   * Amount

2. View all expenses:

   * Shows all recorded expenses
   * Displays total amount spent

3. Exit the application

## Example

Welcome to Expense Tracker

==== MENU ====

1. Add Expense
2. View Expenses
3. Exit

Enter your choice: 1

Enter a date (YYYY-MM-DD): 2026-06-18

Name what you buy: Pizza

Write the amount: 250

Expense added successfully
